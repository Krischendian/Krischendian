### 🔒 AI Investment Research Automation
*Confidential commercial project — code available upon request*

End-to-end AI automation system delivered for a financial 
firm in Richmond Hill, ON. Eliminated 15+ hours/week of 
manual analyst work across 30+ equities.

**What it does:**
- Scheduled daily morning brief — overnight news, company 
  signals, macro overview
- On-demand 6-step sector analysis reports with LLM-ranked 
  relative strength
- Multi-source data pipeline: Bloomberg → FMP → SEC EDGAR 
  with automated fallback
- Finance-specific model routing across Claude and OpenAI

**Key challenge:** Bloomberg data coverage was inconsistent 
across sectors — built a three-tier fallback chain with 
SQLite caching to ensure reliability.

**Stack:** Python · FastAPI · React 19 · TypeScript · 
Claude API · OpenAI API · Bloomberg · FMP · SEC EDGAR
